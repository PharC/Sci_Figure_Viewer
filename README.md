# 🔬 科学期刊图片助手 (Sci-Figure-Viewer)

一款专为科研人员设计的 Chrome 浏览器插件，支持在阅读文献时快速查看大图。

### ✨ 功能特点
- **多平台适配**：支持 Cell, Nature, Science (CNS) 以及 Wiley 系列期刊。
- **高清原图**：点击链接自动提取高清源地址，避免手动缩放。
- **交互体验**：支持侧边栏自由拉伸、图片无级缩放及拖拽。

### 🛠️ 安装方法
1. 下载本项目并解压。
2. 打开 Chrome `chrome://extensions/`，开启 **开发者模式**。
3. 点击 **加载已解压的扩展程序**，选择本目录。
